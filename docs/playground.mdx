## Stateful

<Playground>
  {() => {
    const [counter, setCounter] = React.useState(0);
    return (
      <div>
        <Button onClick={() => setCounter(counter => counter+1)}>
          Increase
        </Button>
        <span style={{ marginLeft : 20 }}>{counter}</span>
      </div>
    )
  }}
</Playground>

## Story wrapper

You can wrap the example code in a `<Story />` component - this will give you an identifier (story id) to reference this specific example of code.

```js:title=playground.mdx
---
name: Playground
menu: Components
---

import { Playground, Story } from '@component-controls/blocks';
import { Button } from '../src/components/Button'

# Button

<Playground>
  <Story name='example'>
    <Button kind="secondary">Click me</Button>
  </Story>  
</Playground>
```
